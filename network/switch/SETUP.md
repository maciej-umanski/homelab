# Setup

## Bootstrap

1. Connect computer with cable to `Ethernet 1 port`
2. Set static IP address on computer to
    ```
       IP: 192.168.88.10
       Mask: 255.255.255.0
    ```
3. Access `http://192.168.88.1` and login using `admin` / `blank password`

## Config

Configure switch manually via SwOS using configuration described in [configuration file](CONFIGURATION.md)
